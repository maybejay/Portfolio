PORTFOLIO WEBSITE TEMPLATE

Thank you for purchasing this portfolio website template!

This is a clean, modern, and responsive portfolio built using
HTML and CSS. It is suitable for students, developers, and freelancers.


FILES INCLUDED
- index.html  → Main website file
- style.css   → Styling and design
- README.txt  → Instructions
- LICENSE.txt → Usage rules


HOW TO USE
1. Open the folder
2. Double-click index.html
3. The website will open in your browser


HOW TO EDIT CONTENT
- Open index.html using:
  - VS Code
  - Notepad++
  - Any code editor

- Change:
  - Name
  - Text
  - Skills
  - Projects
  - Contact info


HOW TO CHANGE COLORS
Open style.css and edit the :root section:
:root {
  --primary: #6366f1;
  --secondary: #8b5cf6;
  --accent: #ec4899;
}


DEPLOYMENT
You can upload this website using:
- GitHub Pages
- Netlify
- Vercel

No backend or database required.


SUPPORT
This product is provided as-is.
Basic HTML/CSS knowledge is recommended.

Enjoy your portfolio!
